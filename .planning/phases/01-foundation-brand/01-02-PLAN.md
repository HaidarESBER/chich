  <name>Task 1: Develop brand name and tagline options</name>
  <files>.planning/BRAND.md</files>
  <action>
    Create .planning/BRAND.md with brand identity research and options.

    **Name criteria:**
    - Works in French and sounds premium
    - Evokes quality/lifestyle, not "smoke shop"
    - Short, memorable, domain-likely-available
    - Avoid cliches (no "hookah heaven", "cloud nine", etc.)

    Generate 3-5 name options with rationale. Consider:
    - French words evoking atmosphere, relaxation, gathering
    - Abstract names that feel premium (like "Everlane" or "Mejuri")
    - Names that work as a lifestyle brand, not just product category

    For each name, include:
    - Name
    - Pronunciation guide (if needed)
    - Meaning/association
    - Domain availability status (check .com and .fr)
    - Tagline suggestion (short, French)

    Structure BRAND.md:
    ```
    # Brand Identity: [Project Name]

    ## Status
    In Development - Pending Decisions

    ## Brand Name Options
    [Options with rationale]

    ## Selected Name
    (To be decided)

    ## Visual Direction
    (Next section)
    ```
  </action>

<task type="auto">
  <name>Task 2: Define visual identity (colors, typography, style)</name>
  <files>.planning/BRAND.md</files>
  <action>
    Update BRAND.md with visual identity based on selected name.

    **Color Palette:**
    Define a restrained palette (5-7 colors max):
    - Primary: Main brand color (likely neutral/sophisticated - charcoal, navy, or warm neutral)
    - Accent: One accent color for CTAs and highlights
    - Background: Off-white or warm white (not stark white)
    - Text: Near-black for readability
    - Muted: For secondary text, borders
    - Success/Error: For UI feedback

    Document each color with:
    - Name
    - Hex value
    - Usage guidelines

    **Typography:**
    Select 2 fonts max:
    - Headings: Modern sans-serif with character (suggest 2-3 options)
    - Body: Clean, highly readable sans-serif

    Criteria: Must be available on Google Fonts or similar free service.

    **Visual Style Guidelines:**
    - Photography direction (lifestyle, product shots - describe the mood)
    - Spacing philosophy (generous whitespace, editorial feel)
    - Icon style (line icons, minimal)
    - Overall mood words (3-5 words defining the aesthetic)

    Add all to BRAND.md under Visual Identity section.
  </action>
  <verify>BRAND.md has complete color palette with hex values, typography choices, style guidelines</verify>
